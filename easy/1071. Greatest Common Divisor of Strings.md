# [1768. Merge Strings Alternately](https://leetcode.com/problems/merge-strings-alternately/description/?envType=study-plan-v2&envId=leetcode-75)

## Intuition
From the problem statement, the phrase “the largest string that divides both strings” suggested that this is a `greatest common divisor (GCD)` problem.<br>
Based on this insight, I approached it by applying the `Euclidean algorithm to strings to find the common pattern.

---

## Approach
1. Ensure that the longer string is placed first; swap the strings if necessary.
2. If the two strings are exactly the same, return that string.
3. If the longer string starts with the shorter string, remove the prefix (the length of the shorter string) from the longer string and recursively apply the same process.
4. If none of the above conditions are satisfied, return an empty string since there is no common divisor string.

---

## Complexity
### Time Complexity
O(n + m), where n and m are the lengths of the two strings.

### Space Complexity
O(n + m), due to recursion and substring operations.

---

## Code
```
class Solution {
    public String gcdOfStrings(String str1, String str2) {
        if (str2.length() > str1.length()) {
            return gcdOfStrings(str2, str1);
        }

        if (str2.equals(str1)) return str1;

        if (str1.startsWith(str2)) {
            return gcdOfStrings(str1.substring(str2.length()), str2);
        }

        return "";
    }
}
```
