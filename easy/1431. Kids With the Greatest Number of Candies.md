# [1431. Kids With the Greatest Number of Candies](https://leetcode.com/problems/kids-with-the-greatest-number-of-candies/description/?envType=study-plan-v2&envId=leetcode-75)

## Intuition
For each child, check whether giving all `extraCandies` to that child would make their total number of candies greater than or equal to the maximum among all children.<br>
To do this, first find the maximum value in the `candies` array, then compare it with each child's potential total.

---

## Approach
1. Iterate through the `candies` array to find the maximum value `maxCandies.
2. Iterate through the array again, and for each child, check if `candies[i] + extraCandies is greater than or equal to `maxCandies`.
3. If the condition is satisfied, add `true` to the result list; otherwise, add `false.
4. Return the result list.

---

## Complexity
### Time Complexity
O(n), where n is the length of the candies.

### Space Complexity
O(n), for the result list.

---

## My Solution
```
import java.util.*;

class Solution {
    public List<Boolean> kidsWithCandies(int[] candies, int extraCandies) {
        List<Boolean> result = new ArrayList<>();
        int[] newCandies = new int[candies.length];
        for (int i = 0; i < candies.length; i++) {
            newCandies[i] = candies[i] + extraCandies;
        }

        for (int i = 0; i < candies.length; i++) {
            int max = Arrays.stream(candies).max().orElse(0);

            if (max <= newCandies[i]) result.add(true);
            else result.add(false);
        }

        return result;
    }
}
```

---

## Review
- The `newCandies array is unnecessary since we can compute `candies[i] + extraCandies` on the fly.
- The maximum value of `candies does not change, so it should be computed only once.
- Calling `Arrays.stream().max()` inside the loop results in redundant computations, increasing time complexity to O(n²).


## Revised Solution
```
import java.util.*;

class Solution {
    public List<Boolean> kidsWithCandies(int[] candies, int extraCandies) {
        List<Boolean> result = new ArrayList<>();
        int maxCandies = -1;
        for (int candy : candies) {
            maxCandies = Math.max(maxCandies, candy);
        }

        for (int i = 0; i < candies.length; i++) {
            if (maxCandies <= candies[i] + extraCandies) result.add(true);
            else result.add(false);
        }

        return result;
    }
}
```
