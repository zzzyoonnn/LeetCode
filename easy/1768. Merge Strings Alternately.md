# [1768. Merge Strings Alternately](https://leetcode.com/problems/merge-strings-alternately/description/?envType=study-plan-v2&envId=leetcode-75)

## Intuition
We iterate through both strings using a single index, based on the length of the longer string.  
At each step, we alternately append characters from `word1` and `word2` if they exist.

The process continues until all characters from both strings are consumed.

## Approach
1. Get the maximum length between `word1` and `word2` as `range`.
2. Initialize a `StringBuilder` to build the result string.
3. Use `index` to iterate from 0 to `range - 1`.
    - If `index` is within `word1`, append `word1.charAt(index)`.
    - If `index` is within `word2`, append `word2.charAt(index)`.
4. Convert `StringBuilder` to a string and return it.

## Complexity
### Time Complexity
O(n), where n is the length of the longer string.

### Space Complexity
O(n), for the output string stored in `StringBuilder`.

## Code
```
class Solution {
    public String mergeAlternately(String word1, String word2) {
        int range = Math.max(word1.length(), word2.length());

        int index = 0;
        StringBuilder sb = new StringBuilder();
        while (index < range) {
            if (index < word1.length()) sb.append(word1.charAt(index));

            if (index < word2.length()) sb.append(word2.charAt(index));

            index++;
        }

        return sb.toString();
    }
}
```
